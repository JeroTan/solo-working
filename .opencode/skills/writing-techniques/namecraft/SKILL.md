# Namecraft

Use this skill when the story needs a name that feels intentional, pronounceable, genre-fit, and connected to the entity's role.

## Research Sources

Use these when internet access is available or when meaning/origin matters:

- Behind the Name: https://www.behindthename.com/
- Behind the Name random generator: https://www.behindthename.com/random/
- Behind the Name meaning search: https://www.behindthename.com/meanings/
- Fantasy Name Generators: https://www.fantasynamegenerators.com/
- Reedsy generators: https://reedsy.com/studio/generators
- Nameberry: https://nameberry.com/
- Etymonline: https://www.etymonline.com/
- Wiktionary: https://www.wiktionary.org/

Use sources for inspiration and checking. Do not copy a distinctive name from an existing modern franchise when it would read as that franchise's property or direct reference.

## Naming Brief

Before suggesting names, identify as many of these as possible:

- **Entity Type:** character, faction, city, nation, school, ability, artifact, monster, title, arc, system, concept, etc.
- **Genre:** fantasy, scifi, romance, mystery, horror, thriller, slice-of-life, litRPG, progression fantasy, etc.
- **Tone:** elegant, brutal, comic, sacred, bureaucratic, cute, uncanny, noble, trashy, ancient, corporate, etc.
- **Sector:** region, class, profession, institution, military branch, cult, guild, noble house, industry, underworld, academy, server, district, etc.
- **Faction:** allegiance, ideology, enemy group, religion, company, kingdom, family, school, race/species, or political bloc.
- **Role / Function:** protagonist, rival, guide, trickster, tyrant, healer, judge, traitor, city of exile, forbidden spell, etc.
- **Personification:** truth, hunger, mercy, ruin, silence, pride, memory, law, chaos, dawn, decay, etc.
- **Materialization:** color, metal, element, planet, flower, weapon, weather, number, animal, mineral, machine, sound, scent, or texture associated with the entity.
- **Humor Role:** serious, deadpan, absurd, cute, pompous, ironic, embarrassing, meme-like, contrast joke, or no humor.
- **Sound Profile:** short/long, hard/soft consonants, vowel-heavy, clipped, ceremonial, modern, archaic, readable in English.
- **Canon Fit:** existing naming patterns, forbidden letters/sounds, similar names to avoid, and glossary terms already used.

## Process

1. **Choose a naming lane**
   - Meaning-forward: name comes from a root word, virtue, flaw, symbol, or myth.
   - Sound-forward: name mainly fits the world/faction's phonetic style.
   - Lore-forward: name reflects history, faction, rank, region, title, or taboo.
   - Humor-forward: name uses irony, contrast, over-formality, understatement, or intentional awkwardness.

2. **Pick source pools**
   - Real personal names: Behind the Name, Nameberry.
   - Fantasy/scifi/place/object names: Fantasy Name Generators, Reedsy.
   - Root words and concepts: Etymonline, Wiktionary.
   - Myth, religion, and occult references: verify meaning and sensitivity before use.

3. **Generate candidates**
   - Give 6-12 options when brainstorming.
   - Mix direct names, softened variants, and invented forms.
   - Include pronunciation if the spelling may be unclear.
   - Include a one-line reason for each candidate.

4. **Check fit**
   - Does it sound like the same world as nearby names?
   - Is it too hard to pronounce or too similar to an existing name?
   - Does the meaning accidentally contradict the character/faction?
   - Is the cultural or religious reference too loaded for casual use?
   - Does the name spoil a twist too early?

5. **Record it**
   - Character names -> `knowledge/character-profiles.md` and `knowledge/character-voices.md`.
   - Factions, titles, systems, places, items, abilities, and terms -> `knowledge/glossary.md`.
   - Locations -> `knowledge/locations.md`.
   - World naming rules -> `knowledge/world-setting.md`.

## Output Format

When suggesting names, use:

```markdown
| Name | Type | Pronunciation | Source / Root | Why It Fits | Caution |
|------|------|---------------|---------------|-------------|---------|
| [Name] | [meaning/sound/lore/humor] | [simple guide] | [root/source] | [fit] | [risk or none] |
```

Then recommend 1-3 strongest options and explain the tradeoff.

## Rules

- Prefer names the user can read aloud without a pronunciation guide.
- Keep one faction's names in the same sound family unless contrast is intentional.
- Use symbolism lightly. A name meaning "truth" for a truth-themed character can work; every name being a direct label feels fake.
- If the user has not chosen, offer options instead of silently locking a major character name.
- For minor throwaway characters, a quick genre-fit name is acceptable.
- For sacred, cultural, or religious references, be careful and explain the weight of the reference.
